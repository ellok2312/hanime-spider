# Hanime!

*_ Hanime!番剧爬虫 _*

使用 Hanime!，你可以轻松的下载你想要的所有里番剧集，并实现类 [`Emby`](https://emby.media/) 的刮削功能

## 当前功能

* 根据输入的番剧名称批量下载所有的剧集
